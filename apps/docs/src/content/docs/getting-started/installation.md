---
title: Installation
description: Install Minions Qa-runs packages
---

## TypeScript / Node.js

```bash
npm install @minions-qa-runs/sdk minions-sdk
```

## Python

```bash
pip install minions-qa-runs
```

## CLI

```bash
npm install -g @minions-qa-runs/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)

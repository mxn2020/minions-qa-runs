# Minions Qa-runs

Test execution records, assertion results, and pass/fail tracking across clawspaces

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-qa-runs/sdk`) — Core library for Node.js
- **Python SDK** (`minions-qa-runs`) — Python bindings
- **CLI Tool** (`@minions-qa-runs/cli`) — Command-line interface
- **Web App** — Interactive playground at [qa-runs.minions.wtf](https://qa-runs.minions.wtf)
